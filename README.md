#  PixelPerk Web UI – Multi-Page Frontend Project

A beginner-level multi-page web application built using **HTML, CSS, and JavaScript**.
The project simulates a simple user flow from a landing page with a login form to a product/download page inspired by UltraEdit.

---

## Features

*  Navigation bar with menu links
*  Search bar UI
*  Landing page with hero section
*  Login form with basic validation
*  Page redirection after login
*  Product/Download page (multi-language download options)
*  Grid-based layout for content display
*  Loader animation before content loads

---

##  Tech Stack

* HTML
* CSS (Flexbox + Grid)
* JavaScript (Vanilla JS)

---

## 🧠 How It Works

1. User lands on the homepage
2. A loader animation is displayed
3. User enters login details
4. Basic validation is performed
5. On success → user is redirected to the download page
6. User can browse and click download options


## ⚠️ Limitations

* No real authentication (frontend only)
* Not fully responsive
* No backend/database integration
* Static content

This project was built as part of my **early web development learning journey** and focuses on understanding UI structure and frontend logic.
